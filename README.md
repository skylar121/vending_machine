### 배운점
1. display: flex 속성 사용하여 전체 레이아웃, 벤딩 머신 내 음료수 레이아웃 구성

2. 여러번 쓰이는 속성의 모듈화 - 선택된 음료수 버튼 부분

3. 품절 버튼 처리
  - 별도의 클래스 부여하여 스타일 속성 적용
  - 가상요소로 품절 이미지 적용 + 커서 선택되지 않도록 함
